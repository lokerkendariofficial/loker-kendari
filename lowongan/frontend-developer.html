Frontend Developer

💰 Gaji: Rp 5.000.000 - Rp 7.000.000 / bulan

📅 Batas lamar: 30 Juni 2026

⏰ Tipe pekerjaan: Penuh waktu (Full Time)

Deskripsi Pekerjaan

PT Teknologi Sultra sedang mencari Frontend Developer yang bersemangat untuk bergabung dengan tim pengembangan produk digital kami. Anda akan bertanggung jawab membangun antarmuka web yang responsif, cepat, dan mudah digunakan.

Kualifikasi

  • Minimal 1 tahun pengalaman dengan React.js atau framework sejenis (Vue/Angular).
  • Menguasai HTML5, CSS3, dan JavaScript (ES6+).
  • Pernah menggunakan Git untuk version control.
  • Memahami prinsip responsif design dan aksesibilitas web.
  • Bekerja sama dengan tim backend melalui REST API.
  • Diutamakan yang memiliki portofolio atau GitHub aktif.

Tanggung Jawab

  • Mengembangkan fitur baru pada aplikasi web perusahaan.
  • Memastikan performa halaman optimal (Core Web Vitals).
  • Melakukan debugging dan perbaikan bug.
  • Berkolaborasi dengan desainer UI/UX.

Cara Melamar

Kirimkan CV dan portofolio Anda ke email: hrd@ptteknologisultra.com dengan subjek "Frontend Developer - [Nama Anda]".
